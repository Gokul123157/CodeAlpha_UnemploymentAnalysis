# Unemployment Analysis with Python

## Objective
This project analyzes unemployment rates in India using Python.

## Features
- Data Cleaning
- Data Exploration
- Trend Visualization
- Covid-19 Impact Analysis
- State-wise Unemployment Analysis

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Conclusion
The project identifies unemployment trends and highlights the impact of Covid-19 on employment levels across different states.
